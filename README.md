# Installing dependencies
```
npm install discord.js --save
```

# Setup
Create a config.json file with this content :
```jsonc
{
  "prefix": "!", // You can change the prefix to whatever you want it doesn't have to be - !
  "token": "Your-Bot-Token"
}
```
